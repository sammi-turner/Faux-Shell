# Faux-Shell

A faux unix shell in the browser with a faux persistent file system.

<br>

## Requirements

```
node
npm
```

<br>

## Install node modules

```
npm i
```

<br>

## Run the dev server

```
npm run dev
```

<br>

## Build the dist folder for deployment

```
npm run build
```
